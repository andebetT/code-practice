# code-practice


This repository contains my solutions and experiments for coding practice problems.  
It’s focused on **data structures**, **algorithms**, and **string/array manipulation**—implemented mostly in **Python**.

---

## 📁 Project Structure

- **Python files / scripts**: Standalone solutions for individual problems.
- Each problem typically includes:
  - the core function implementation
  - a small example / test snippet

> Note: Some scripts currently include print statements and example runs directly in the file.

---

## ✅ Included Topics / Techniques

### Strings
- Palindrome checking (two-pointer approach)
- Constructing the **lexicographically smallest palindrome** using character counts

### Arrays & Sliding Window
- Maximum of every subarray of length `k` (monotonic deque)
- Minimum of every subarray of length `k` (monotonic deque)

### Sorting / Intervals
- Merging-like logic to compute total “covered” difference across intervals

### Peak Finding (Binary Search style)
- Finding the index/value of a **peak element** with handling for duplicates

### Special Array/DP Style Problems
- Cost-minimization using greedy/DP-style transitions (e.g., swap/delete cost problems)

---

## 🧠 Algorithm Patterns Used

- **Two Pointers**
- **Frequency counting** (`collections.Counter`)
- **Sorting + Reconstruction**
- **Monotonic Deque** (sliding window min/max)
- **Binary Search with edge-case handling**
- **Greedy/DP cost transitions**
